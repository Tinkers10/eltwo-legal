Blind-Spot Pro Privacy Policy

Last updated: June 10, 2026

Blind-Spot Pro is local-first WCAG contrast software for Chrome. It inspects page colors and computed styles locally so users can find contrast issues, review suggestions, and export accessibility reports.

What Blind-Spot Pro Stores Locally

Blind-Spot Pro stores settings, contrast history, favorite color pairs, report/export preferences, and cached Gumroad license entitlement details in Chrome storage on your device.

What Leaves Your Device

When you activate or re-check a Pro license, Blind-Spot Pro sends the license key to Gumroad's license verification API. Gumroad handles checkout and payment externally. el-two LLC does not receive payment-card details through the extension.

Website Access

Blind-Spot Pro reads page colors, computed styles, and selected elements only to calculate WCAG contrast and generate local reports. Page content is not sold, collected for analytics, or transmitted to el-two LLC.

Remote Code

Blind-Spot Pro does not load remote executable code. All JavaScript and CSS used by the extension is bundled in the Chrome extension package.

Contact

Questions can be sent to info@el-two.com.
